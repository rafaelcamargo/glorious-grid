introdução

Esse grid foi feito para front enders que curtem simplicidade e que não curtem a maneira com a qual Bootstrap e Foundation tratam seus grids. Diferentemente deles, esse grid não requer várias classes css de acordo com o tamanho da tela, mantendo a marcação limpa.

instalar

Você pode instalá-lo usando npm ou pode, se não manjar de gerenciamento de dependências, baixar o código e inserí-lo manualmente:

npm install ggrid --save
