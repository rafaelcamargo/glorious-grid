Glorious Grid

This is a worryless-responsive grid for developers who don't want to have to choose several columns sizes depending on size of screen. So, Glorius Grid has no those famous classes used by bootstrap and foundation like small or col-sm, medium or col-md and large or col-lg.

Usage

When using Glorius Grid, columns can be defined in just three ways. Let the grid becomes single column for screens with a lower width than your grid container gg-container, keep the same # of columns for medium screen size or keep the same # of columns for all screen sizes.

You you want to let grid becomes single columns, you don't need to do anything. It's the default behaviour of Glorious Grid.
Otherwise, you can use the following classes in a specific row gg-row:
gg-keep-on-md on respective gg-keep-on-sm containing these columns.

Single Column Grid

The default behaviour is the grid to become single grid when the screen width is smaller than gg-container width.
